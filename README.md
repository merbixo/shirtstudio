# Merbi's Anime Collection — V2 (GitHub Pages ready)

This is a static demo site (TailwindCSS) built for GitHub Pages. It includes:
- Printify placeholder links (replace with your actual Printify/Shopify links)
- "Coming Soon" animated placeholders
- Spy x Family preview SVG (placeholder image)
- Notes about using Wise for payouts (static info modal)

## Files
- index.html — main page
- assets/styles.css — custom styles
- assets/script.js — JS handlers
- images/spy-family-design1.svg — placeholder product image

## Quick local preview
1. Open `index.html` in your browser.
2. OR serve with `python -m http.server` inside the folder and visit http://localhost:8000
